# RD-55 "Flylens75"

**Drone Designator**: RD-55
**Code Name**: Flylens75
**Operator**: SpyD
**Platform**: 75mm Ducted Whoop

This repository serves as the source of truth for the RD-55 "Flylens75" drone - a Flywoo FlyLens 75 2S with Walksnail V1.3 and ELRS 2.4GHz.
